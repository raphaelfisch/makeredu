# MakerEdu

Interaktive Beurteilungstools für den Making-Unterricht an Schweizer Schulen.

🌐 [**makeredu.raphaelfisch.github.io/makeredu**](https://raphaelfisch.github.io/makeredu/)

\---

## Tools

### 📄 Beurteilungs-Generator

Erstellt individuell konfigurierbare Beurteilungsbögen als PDF. Lehrpersonen wählen Kompetenzbereiche und Indikatoren aus, ergänzen den Auftragstext und exportieren einen druckfertigen Bogen.

→ [Zum Generator](https://raphaelfisch.github.io/makeredu/makeredu_generator.html)

### 📊 Tracker

Erfasst und visualisiert Schüler:innen-Beurteilungen über mehrere Lektionen hinweg. Importiert die Indikatorenauswahl direkt aus dem Generator und zeigt Entwicklungen per Sparklines und Verlaufsgraphen.

→ [Zum Tracker](https://raphaelfisch.github.io/makeredu/makeredu_tracker.html)
